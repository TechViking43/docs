This quickstart walks you through everything you need to go from zero to a running QuantumFlow AI workflow. By the end, you'll have a verified account, a live API key, and a workflow you created and triggered yourself — all in about five minutes. No prior setup is required.

## Steps

<Steps>
  <Step title="Create Your Account">
    Go to [app.quantumflow.ai](https://app.quantumflow.ai) and click **Sign Up**. Enter your name, work email address, and a password, then click **Create Account**.

    QuantumFlow AI will send a verification email to the address you provided. Open it and click **Verify Email** to activate your account. Once verified, you'll land directly in your dashboard and be ready to proceed.
  </Step>
  <Step title="Get Your API Key">
    From your dashboard, navigate to **Settings → API Keys** in the left sidebar. Click **Create New Key**, give it a descriptive name (for example, `quickstart-key`), and click **Generate**.

    Copy the key immediately and store it somewhere secure — such as a password manager or an environment variable in your local shell. QuantumFlow AI will only display the full key once.

    <Note>
      Your API key is a secret credential. Never hard-code it in client-side code, commit it to a public repository, or share it in plain text. Use environment variables or a secrets manager to keep it safe.
    </Note>
  </Step>
  <Step title="Make Your First API Call">
    With your API key in hand, create your first workflow by sending a `POST` request to the `/v1/workflows` endpoint. Replace `YOUR_API_KEY` with the key you just generated.

    <CodeGroup>
      ```bash curl
      curl -X POST https://api.quantumflow.ai/v1/workflows \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{"name": "My First Workflow", "description": "Automates my daily reports"}'
      ```

      ```python Python
      import requests

      response = requests.post(
          "https://api.quantumflow.ai/v1/workflows",
          headers={"Authorization": "Bearer YOUR_API_KEY"},
          json={"name": "My First Workflow", "description": "Automates my daily reports"}
      )
      print(response.json())
      ```

      ```javascript JavaScript
      const response = await fetch("https://api.quantumflow.ai/v1/workflows", {
        method: "POST",
        headers: {
          "Authorization": "Bearer YOUR_API_KEY",
          "Content-Type": "application/json"
        },
        body: JSON.stringify({ name: "My First Workflow", description: "Automates my daily reports" })
      });
      const workflow = await response.json();
      console.log(workflow);
      ```
    </CodeGroup>

    A successful request returns a `201 Created` response with the new workflow object:

    ```json
    {
      "id": "wf_01j9abc123",
      "name": "My First Workflow",
      "description": "Automates my daily reports",
      "status": "draft",
      "created_at": "2024-01-15T10:30:00Z"
    }
    ```

    The workflow starts in `draft` status, meaning it has been created but not yet activated. You'll activate it in the next step.
  </Step>
  <Step title="View Your Workflow in the Dashboard">
    Head back to [app.quantumflow.ai](https://app.quantumflow.ai) and open the **Workflows** section from the sidebar. You'll see **My First Workflow** listed there with a `Draft` badge.

    Click the workflow name to open the Workflow Builder, where you can add triggers, connect AI agents, and wire up actions. When you're ready to run it, toggle the workflow to **Active** — QuantumFlow AI will begin executing it according to your configured triggers and schedule.

    You can monitor every execution in the **Task Logs** tab, where each run is recorded with its status, duration, and full output.
  </Step>
</Steps>
